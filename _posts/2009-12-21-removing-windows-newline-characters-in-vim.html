--- layout: post title: Removing Windows Newline Characters in Vim excerpt: How to break up the ^M party. categories: - Linux - Programming tags: - Linux - Vim status: publish type: post published: true --- Sometimes when working on projects in both Linux and Windows, a file will end up with Windows newline characters, which show up as ^M in Vim. Annoying as this is, there's a quick fix to remove them in Vim using a regular expression replace: 
:%s///g
Magically, all the ugly Windows newline characters are gone.
