That was quite awhile ago, and since then, git, and the git hosting platform GitHub, have become quite awesome. GitHub has an intuitive, beautiful user interface, with fantastic issue tracking, merge requests and overall repository browsing and viewing. Git itself has extremely fast and simple branching built it, as well as just about any other feature Subversion has, but done better. Because of these reasons, I have moved the stjerm project from Google Code to GitHub. You can find it here: https://github.com/mrkrstphr/stjerm. Feel free to check it out, submit bugs/features or fork it and do a merge request of your own features!
